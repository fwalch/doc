31 /*
32  * When adding an Ex command:
33  * 1. Add an entry in the table below. Keep it sorted on the shortest
34  * version of the command name that works. If it doesn't start with a
35  * lower case letter, add it at the end.
36  * 2. Add a "case: CMD_xxx" in the big switch in ex_docmd.c.
37  * 3. Add an entry in the index for Ex commands at ":help ex-cmd-index".
38  * 4. Add documentation in ../doc/xxx.txt. Add a tag for both the short and
39  * long name of the command.
40  */
